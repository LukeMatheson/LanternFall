In order to run locally: 

1. download and unzip into folder of your choosing
2. open command prompt into and cd into: {chosen folder}/team-lijoi/LanternFall/app
3. run "npm install" command
4. run "node server.js" command
5. you're all set! navigate to http://localhost or lanternfall.com to see our project.
